LLM是无状态的纯函数，需要依赖prompt来维持状态（状态管理）
LLM只能输入，输出文本，需要通过解析输出来影响外部环境

LLM/Tools



状态管理模块
{
    通过与LLM，Tool交互的过程中，不断改变状态
    状态里面有什么{
        system prompt
        message memory{
            user message
            llm message
            tool message
        }
        tool list
    }

}
